# Wordpress plugin Development with wp-scripts
Quick Start Guide
1. Prerequisites
```sh
# Node.js 18+ and npm
node --version  # Should be v18+
npm --version
```

2. Create Plugin Structure
```sh
# Create plugin directory
mkdir -p wp-content/plugins/my-awesome-plugin
cd wp-content/plugins/my-awesome-plugin

# Initialize npm project
npm init -y

# Install @wordpress/scripts
npm install --save-dev @wordpress/scripts

# Install WordPress dependencies (if needed)
npm install --save-dev @wordpress/element @wordpress/components @wordpress/i18n
```

3. Project Structure
```sh
my-awesome-plugin/
├── src/
│   ├── index.js          # Main entry point
│   ├── style.scss        # Frontend styles
│   └── editor.scss       # Editor styles
├── build/                # Generated by wp-scripts (gitignored)
├── includes/
│   └── class-plugin.php  # PHP logic
├── languages/            # Translation files
├── my-awesome-plugin.php # Main plugin file
├── package.json
├── .gitignore
└── README.md
```

Starts the build for development.
```sh
npm start
```

Builds the code for production.
```sh
npm run build
```

Formats files.
```sh
  $ npm run format
```

Lints CSS files.
```sh
npm run lint:css
```

Lints JavaScript files.
```sh
npm run lint:js
```

Creates a zip file for a WordPress plugin.
```sh
npm run plugin-zip
```

Updates WordPress packages to the latest version.
```sh
npm run packages-update
```

To enter the directory type:
```sh
cd wp-plugin-contact
```
You can start development with:
```sh
npm start
```

You can start WordPress with:
```sh
npx wp-env start
```
